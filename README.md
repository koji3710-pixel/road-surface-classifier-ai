# Road Surface Condition Classifier (ResNet18)

車載カメラ画像から路面状況（積雪・雨・霧・夜）を判定するAIプロトタイプです。
単なる精度の追求にとどまらず、**「説明可能AI（XAI）」を用いたモデルの妥当性検証**と、**「実運用を見据えた環境最適化」**に注力しました。

## 🌟 プロジェクトのハイライト
- **データ最適化**: 生データ7GBを、精度を維持したまま40MBまで軽量化し、学習効率を大幅に改善。
- **汎化性能の向上**: 複雑なデータ拡張（GaussianBlur等）により、フロントガラスの汚れ等のノイズに対するロバスト性を獲得。
- **XAIによる検証**: Grad-CAMを用い、AIが「路面のテクスチャ」を根拠に判断していることを可視化・実証。
- **コンテナ化**: Dockerを採用し、環境に依存しない推論実行環境を構築。

## 🔍 Grad-CAMによる失敗分析と改善
初期モデルでは、地吹雪を「霧（Fog）」と誤判定する課題がありました。
Grad-CAMで解析した結果、AIが路面ではなく「画面全体の明るさ（白さ）」に過剰適応している（ショートカット学習）ことが示唆されました。
データ拡張によりこのバイアスを排除し、最終的に路面のタイヤ痕等の直線的な特徴を捉えるモデルへと改善しました。
